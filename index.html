VPS v.s Shared Web Hosting

簡單來說
VPS是一台獨立的實體伺服器一樣，可以在上面安裝OS或程式…等;
Shared Hosting是單純從一台實體CPU切出來，所以會跟其他人共用到RAM;

價格 服務類型 穩定度 IP 操作自由度 相關知識需求

(例如cPanel)		 低，按圖文操作即可
